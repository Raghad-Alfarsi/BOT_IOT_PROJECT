Bot-IoT Data Cleaning & EDA

Overview

This project focuses on cleaning and exploring the Bot-IoT dataset to understand IoT network traffic and prepare the data for analysis.

Work Done
	•	Data inspection and understanding
	•	Exploratory Data Analysis (EDA)
	•	Data preprocessing and cleaning
	•	Exporting the cleaned dataset

Streamlit App

A simple Streamlit app is included to visualize key EDA plots.

streamlit run app.py

Files
	•	app.py – Streamlit app
	•	Raghad_BOT_IOT_PROJECT.ipynb – Data cleaning & EDA
	•	cleaned_data.zip – Cleaned dataset
	•	IDS_Project_Report.pdf – Project report
